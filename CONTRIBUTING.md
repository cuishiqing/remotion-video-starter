# Contributing to Remotion Video Starter

感谢你对 Remotion Video Starter 的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果你发现了 bug 或有功能建议：

1. 检查 [Issues](https://github.com/cuishiqing/remotion-video-starter/issues) 是否已有类似问题
2. 如果没有，创建新的 Issue，详细描述：
   - 问题描述
   - 复现步骤
   - 预期行为
   - 实际行为
   - 环境信息（Claude Code 版本、操作系统等）

### 提交代码

1. **Fork 仓库**
   ```bash
   # 在 GitHub 上点击 Fork 按钮
   ```

2. **克隆你的 Fork**
   ```bash
   git clone https://github.com/YOUR-USERNAME/remotion-video-starter.git
   cd remotion-video-starter
   ```

3. **创建分支**
   ```bash
   git checkout -b feature/your-feature-name
   # 或
   git checkout -b fix/your-bug-fix
   ```

4. **进行修改**
   - 遵循现有的代码风格
   - 更新相关文档
   - 测试你的修改

5. **提交修改**
   ```bash
   git add .
   git commit -m "feat: add xxx feature"
   # 或
   git commit -m "fix: resolve xxx issue"
   ```

6. **推送到 GitHub**
   ```bash
   git push origin feature/your-feature-name
   ```

7. **创建 Pull Request**
   - 在 GitHub 上创建 PR
   - 详细描述你的修改
   - 引用相关的 Issue

## 开发指南

### 项目结构

```
remotion-video-starter/
├── SKILL.md           # Skill 核心文档
├── README.md          # 项目说明
├── LICENSE            # MIT 许可证
└── assets/            # 资源文件
    └── social-preview.svg
```

### Commit 规范

使用语义化提交信息：

- `feat:` 新功能
- `fix:` Bug 修复
- `docs:` 文档更新
- `style:` 代码格式调整
- `refactor:` 代码重构
- `test:` 测试相关
- `chore:` 构建/工具相关

**示例：**
```
feat: add support for landscape video format
fix: resolve audio synchronization issue
docs: update installation instructions
```

### 代码风格

- 使用 Markdown 格式
- 中文为主，关键术语保留英文
- 代码示例使用语法高亮
- 保持简洁清晰

## 测试

在提交 PR 前，请确保：

1. ✅ Skill 文档语法正确
2. ✅ 所有链接可访问
3. ✅ 代码示例可以运行
4. ✅ 文档更新完整
